# 📱 Mobile Responsive Testing Checklist - Nova Hospital Website

## ✅ Desktop to Mobile Testing Guide

### **How to Test Mobile View:**
1. Open the website in your browser
2. Press `F12` or right-click → "Inspect Element"
3. Click the mobile/tablet icon in developer tools
4. Select different device sizes (iPhone, iPad, Android)
5. Test both portrait and landscape orientations

---

## 🔍 **Mobile Responsive Features to Test:**

### **1. Header & Navigation**
- [ ] Mobile hamburger menu appears on screens ≤ 768px
- [ ] Clicking hamburger opens full-screen navigation overlay
- [ ] Navigation slides in smoothly from left
- [ ] Emergency info is hidden on mobile
- [ ] Logo scales appropriately
- [ ] Clicking outside nav closes it
- [ ] Swipe left gesture closes navigation

### **2. Hero Section**
- [ ] Hero image displays properly on mobile
- [ ] Title text is readable (not too large/small)  
- [ ] CTA buttons stack vertically and fill width
- [ ] Trust indicators wrap nicely
- [ ] Floating stats are hidden on mobile (cleaner layout)
- [ ] Hero stats bar shows 2x2 grid on mobile, 1x4 on small screens

### **3. Services Section**
- [ ] Service cards display in single column on mobile
- [ ] Cards maintain equal height (~300px minimum)
- [ ] All 4 service cards are visible and accessible
- [ ] Service icons scale down appropriately
- [ ] Text remains bold and readable

### **4. Statistics Section**
- [ ] Stat cards display in single column

### **5. Testimonials Section**

- [ ] Contact cards stack vertically
